Relatório Técnico de Engenharia

Especificações da PWA e Aplicação SSR | Next.js 14

Arquitetura de Software

O projeto foi desenvolvido sob a arquitetura App Router do Next.js 14, utilizando React 18 e Server Components como padrão para maximizar performance no tempo de carregamento da página (TTFB).

Design System e Estilização

Todo o controle de tokens CSS foi mapeado num arquivo single-source-of-truth via TailwindCSS (v3), garantindo que não haja drift visual nas cores Nude/Rose #B08D8D ou Off-white #FAF9F7. Animações orquestradas nativamente usando Framer Motion (AnimatePresence, variants), mantendo a fluidez a 60FPS.

Otimizações Aplicadas

  • Fontes dinâmicas Otimizadas: O uso do pacote next/font/google previne Cumulative Layout Shift (CLS).
  • Prevenção de FOUC: Resolvido "Flash of Unstyled Content" e "Hydration Error" gerenciando client render vs SSR headers no layout base.
  • Roteamento Dinâmico Anchor: Módulo Tratamentos programado para mapear dados em tempo real da "fonte da verdade" lib/content.ts, tornando fácil expandir novos serviços.
